Kelowna International Airport

KelownaBCCanada

With a few exceptions, airport codes starting with ‘Y’ designate Canadian airports. The ‘LW’ comes from the city’s name: Kelowna. The phrase ‘Your Link to the World’ (a backronym) has been used in the YLW branding to make the code more memorable.
